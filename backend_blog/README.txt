Проектная работа третьего спринта.
В папке backend_blog создан бэкенд приложения-блога с использованием Spring Framework на базе Tomcat 10.
Фронтенд работает на веб-сервере Nginx (выдан готовый проект),
который принимает запросы от браузера пользователя на http://localhost:80/,
отправляет запросы на получение данных в бэкенд по REST.
Бэкенд работает в сервлет-контейнере Tomcat, принимает REST-запросы от фронтенда на http://localhost:8080/,
отправляет SQL-запросы на создание/получение/изменение/удаление данных в БД MySQL, формирует ответ и возвращает фронтенду.
Скрипт на создание схемы БД находится по пути ./db.
